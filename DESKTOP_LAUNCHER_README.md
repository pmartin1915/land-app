# Alabama Auction Watcher - Desktop Launcher Integration A comprehensive desktop launcher system that transforms the command-line Alabama Auction Watcher into a user-friendly desktop application with double-click launch capabilities. ## Overview This desktop launcher integration provides: - **Cross-platform compatibility** (Windows, macOS, Linux) - **One-click application launching** via desktop shortcuts - **Smart GUI launcher** with real-time monitoring - **System tray integration** for background operation - **AI monitoring integration** with existing frameworks - **Automated installation system** for desktop shortcuts - **Comprehensive health checking** and diagnostics ## Directory Structure ``` launchers/ ├── windows/ # Windows batch launchers │ ├── launch_main_app.bat │ ├── launch_backend_api.bat │ ├── launch_enhanced_dashboard.bat │ └── health_check.bat ├── macos/ # macOS command launchers │ ├── launch_main_app.command │ ├── launch_backend_api.command │ └── launch_enhanced_dashboard.command ├── linux/ # Linux desktop integration │ ├── alabama-auction-watcher.desktop │ ├── alabama-auction-api.desktop │ └── launch_scripts/ │ ├── launch_main_app.sh │ ├── launch_backend_api.sh │ ├── launch_enhanced_dashboard.sh │ └── health_check.sh └── cross_platform/ # Cross-platform Python tools ├── smart_launcher.py # Main GUI launcher ├── system_tray.py # System tray integration └── ai_integration.py # AI monitoring integration icons/ # Application icons ├── main_app.ico ├── backend_api.ico ├── enhanced_dashboard.ico ├── health_check.ico └── README.md installer/ # Installation tools ├── setup_desktop_integration.py ├── create_shortcuts.py └── installation_config.json ``` ## Quick Start ### Option 1: Simple Setup (Recommended) 1. **Create desktop shortcuts:** ```bash python installer/create_shortcuts.py ``` 2. **Launch the Smart GUI Launcher:** ```bash python launchers/cross_platform/smart_launcher.py ``` 3. **Use desktop shortcuts** to launch components directly ### Option 2: Full Desktop Integration 1. **Run the comprehensive installer:** ```bash python installer/setup_desktop_integration.py ``` 2. **Enable optional features** (auto-startup, file associations) 3. **Access from desktop** or start menu/applications folder ## Platform-Specific Usage ### Windows -
